description=As an [b]admin, a host, or in debug[/b] right click any object on the map - you should see context menus for "assign store".
description=
description=When assigning a store you can 'connect' that object to an existing store or copy an existing store - when first starting your setup the only option is to 'copy' the 'blank' template.
description=
description=As a player you will need to craft a deed (if this option is enabled on your server/save.) The deed recipe is generated from a sandbox option so it may differ.
description=
description=Typing into the red fields on the bottom right will decide on the item type. You can also search by display category if 'is category' is selected. The other options to decide on are "cost", "quantity", "resell %", and if the item will be resold.
description=
description=Once you are ready to list your item select + to add it. Trying to add the same type/category will overwrite the old one.
